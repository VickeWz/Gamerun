Fakta om appen

Vi på Game Run vill att det ska finns en app som både är bra för din hälsa och ska vara rolig. Alltså vårat ultimata mål är att denna app ska göra så att barn och unga ska tycka att det är roligt att röra sig. Studier har visat att den nya generationen av barn rör sig alldeles för lite och föräldrar har svårt att få de att röra på sig. Det är just där som Game Run hjälper till, eftersom föräldrarna kan till exempel säga" om du ska spela mer så måste du ut och röra dig". Alltså appen hjälper vuxna att motivera sina barn.



Just nu så verkar Game Run ha en väldigt ljus framtid, det finns många nya och kreativa idéer. Alla grundare verkar vara väldigt positiva med att göra idéen Game Run en verklighet. Det finns också idéer kring att ge Game Run någon typ av spel värld, men inget är bestämt. Därför är Game Run en bra möjlighet som kommer förändra ditt liv, på det sättet att rörelse gör så att du mår bättre och du kommer orka mer i vardagen


Game Run kommer göra din vardag och ditt 'liv' roligare på ett eller annat sätt. Om du rör sig kommer din kropp må bättre rent psykiskt och du kommer bli bland annat motiverad till att hitta på saker, som du kanske inte hade varit om du inte orkat.
